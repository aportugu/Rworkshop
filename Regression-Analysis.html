Regression analysis

Regression analysis is a statistical method used to examine the relationship between one or more independent variables (predictors) and a dependent variable (outcome). Its primary goal is to understand how the dependent variable changes as the independent variables vary.

Regression analysis seeks to model the relationship between the variables, typically by fitting a mathematical equation to the observed data. This equation can then be used to make predictions or infer the strength and direction of the relationships between variables.

2. Learn how to perform logistic regression analysis

Logistic regression is a statistical method used for modeling the relationship between a binary outcome variable and one or more predictor variables. The dependent variable is binary, meaning it takes on only two possible values, often coded as 1 and 0. These values could represent categories like success and failure, yes and no, or positive and negative.

The goal of logistic regression is to model the probability that the binary outcome variable is equal to 1 (or takes on one of the categories) as a function of the predictor variables. Logistic regression uses the logistic function (also known as the sigmoid function) to model the relationship.

The logistic regression model can be mathematically represented as:

\(logit(p) = log(\frac{p}{1-p}) = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... +\beta_kx_k\)

Where:

  • \(p\) is the probability of the outcome variable being 1
  • \(x_1, x_2, x_3,...,x_k\) are the predictor variables
  • \(\beta_0, \beta_1,...\beta_k\) are the coefficients (parameters) to be estimated
  • \(logit(p)\) is the natural logarithm of the odds of the outcome variable being 1

The coefficients estimated by logistic regression indicate the strength and direction of the relationship between each predictor variable and the log odds of the outcome variable being 1. These coefficients can be exponentiated to obtain odds ratios, which represent the multiplicative change in the odds of the outcome variable for a one-unit change in the predictor variable.

3. Learn how to perform linear regression

Linear regression is a statistical method used to model the relationship between a dependent variable (often denoted as \(Y\)) and one or more independent variables (often denoted as \(X_1, X_2,...X_k\)). It assumes that the relationship between the dependent variable and the independent variables is linear in nature.

The linear regression model can be mathematically represented as:

\(Y = \beta_0 + \beta_1X_1 + \epsilon\)

Where:

  • \(Y\) is the dependent variable (also called the response variable or outcome)
  • \(X_1\) is the independent variables (also called the predictor variable)
  • \(\beta_0\) is the intercept, representing the value of \(Y\) when \(X_1\) is zero
  • \(\epsilon\) is the error term, representing the random variability or noise in the relationship not explained by the model

In a multivariate linear regression model with more than one independent variable, the equation expands to:

\(Y = \beta_0 + \beta_1X_1 + \beta_2X_2+...+\beta_kX_k + \epsilon\)

Where \(X_1, X_2,...,X_k\) are the independent variables, and \(\beta_1, \beta_2,...,\beta_k\) are the corresponding coefficients.

The primary goal of linear regression is to estimate the values of the coefficients that minimize the difference between the observed values of the dependent variable Y and the values predicted by the linear model. This is typically achieved using the method of least squares, which minimizes the sum of the squared differences between the observed and predicted values.

4. Learn how to perform Cox regression.

Cox regression, also known as Cox proportional hazards regression, is a statistical method used for modeling survival time data. It is a semi-parametric model that does not make specific assumptions about the shape of the hazard function (the instantaneous rate of event occurrence over time). It estimates the hazard function using partial likelihood estimation, allowing for the assessment of the effect of predictor variables on the hazard rate while accounting for censoring in the data.

In Cox regression, the hazard function \(h(t)\) at any time \(t\) for an individual \(i\) is modeled as the product of a baseline hazad function \(h_0(t)\) and the exponential function of a linear combination of predictor variables:

\(h(t)_i = h_0(t) \times exp(\beta_1X_{i1} + \beta_2X_{i2} + ... + \beta_kX_{ik})\)

Where:

  • \(h(t)_i\) is the hazard function for individual \(i\) at time \(t\)
  • \(h_0(t)\) is the baseline hazard function, representing the hazard rate when all predictor variables are zero
  • \(X_{i1} + X_{i2} + ... + X_{ik}\) are the predictor variables for individual \(i\)
  • \(\beta_1 + \beta_2 + ... + \beta_k\) are the coefficients (parameters) to be estimated, representing the log hazard ratios associated with the predictor variables

The key assumption of Cox regression is the proportional hazards assumption, which states that the hazard ratios (the relative hazard rates) associated with each predictor variable are constant over time. This assumption implies that the hazard curves for different levels of the predictor variables are proportional to each other, although their baseline hazard functions may differ.

Cox regression allows for the estimation of hazard ratios, which quantify the relative change in the hazard rate associated with a one-unit change in the predictor variable. Additionally, Cox regression can accommodate censoring in the data, where individuals are followed for a finite period, and some may not experience the event of interest by the end of the study.

4c. Check whether the proportional hazards assumption is violated.

The proportional hazards assumption states that the hazard ratio associated with a predictor variable is constant over time. This means that the relative effect of the predictor variable on the hazard of an event occurring remains the same throughout the study period.

The Schoenfeld residual is a measure of the difference between the observed and expected values of the predictor variable at each time point. It quantifies how much a predictor variables effect changes over time. A positive Schenfeld residual indicates thet the predictor variables effect is higher than expected at that time, while a negative Schoenfeld residual indicates that the effect is lower than expected.

The scaled Schoenfeld residual is a standardized version of the Schoenfeld residual, which is divided by its estimated standard error. The standardization helps in comparing the magnitude of the residuals across different predictor variables and assessing their significance.

The scaled Schoenfeld residuals can be plotted against time to visually inspect whether there is any systematic pattern or trend. A significant departure from zero or a noticeable pattern in the plot suggests a violation of the proportional hazards assumption for that predictor variable.

The output of the cox.zph() function provides a test for each predictor variable in the model, evaluating whether the assumption of proportional hazards is violated. The test typically includes a chi-square statistic and a p-value. If the p-value is below a chosen significance level (e.g., 0.05), it suggests that the proportional hazards assumption may be violated for that predictor variable.

This output can be fed into the plot() and print() functions to visualize the scaled Schoenfeld residuals over time and see the chi square statistic/p-values, respectively.

Show code 
res.cox <-coxph( Surv(Days.to.relapse.death.or.DLC, Relapse.or.death) ~ CAR.T.Product.Type +  Largest.lesion + LDH.preLD + ALC.preLD + Age + Sex + HCT.CI, db)

## Testing proportional hazards assumption
